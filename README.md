# OneTicket Skills

Agent skills for the [OneTicket](https://github.com/dsissoko/oneticket-core) multi-agent framework.

## Installation

```bash
apm install dsissoko/oneticket-skills
```

## Skills

Skills are located in `.agents/skills/`. This catalog is under active development — v1.0.0 skills coming soon.

## Contributing

Skills are authored following the OneTicket skill authoring standards.
See [oneticket-core](https://github.com/dsissoko/oneticket-core) for details.
